= custom_improvements
Установка
==============================================================================================
1) выполнить rake redmine:plugins:migrate NAME=custom_improvements RAILS_ENV=production
2) настроить плагин через меню: Администрирование => Модули => На против
   плагина SunStrike Redmine Custom Improvements нажать параметры, выбрать настройки и нажать "Применить".
ссылка на настройки: http(s)://<адрес сервера>/settings/plugin/custom_improvements
